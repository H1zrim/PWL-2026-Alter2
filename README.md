# PWL-MVC

Project **Mini MVC PHP** untuk kebutuhan pembelajaran `PWL`, dengan tampilan landing page bertema **mint & matcha** menggunakan **Bootstrap 5 CDN**.

## Fitur

- Struktur folder MVC sederhana
- Routing dasar untuk halaman `Home`, `About`, dan `Contact`
- Shared layout dengan `navbar` dan `footer`
- Landing page welcome yang responsif
- Styling modern dengan nuansa warna mint dan matcha

## Struktur Singkat

```text
app/
├── Controllers/
├── Core/
└── Views/
    ├── home/
    ├── layouts/
    └── partials/
```

## Menjalankan Project

1. Simpan project di folder `htdocs` XAMPP.
2. Pastikan **Apache** pada XAMPP sudah berjalan.
3. Buka browser dan akses:

```text
http://localhost/PWL/PWL-MVC/public
```

## Teknologi

- PHP
- Bootstrap 5 CDN
- XAMPP
